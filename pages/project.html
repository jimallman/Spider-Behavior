Arachnolingua by pmidford

This is the developers page for Arachnolingua, an ontology-based vocabulary of terms used in published descriptions of spider behavior. If you are here you are probably looking to either:
  1. Just want to know how the ontology update and generation process works
  2. Fork a copy to start behavior term coverage in a different taxonomic group
  3. Fork a copy because you think the existing assignments are irrepairably wrong
As being successful in items 2 or 3 require understanding item 1, this page will focus on that. If you're just interested in the data, how to use it or add to it, you should go back to the main page.
